# 📊 Dashboard de Vendas no Excel

## 📌 Descrição do Projeto
Este projeto consiste na criação de um dashboard de vendas no Excel com o objetivo de analisar o desempenho comercial de forma visual e intuitiva.

O dashboard permite:
- Acompanhar receita total
- Analisar vendas por período
- Identificar produtos mais vendidos
- Visualizar desempenho por região/vendedor

## 📂 Estrutura do Repositório
- `dashboard_vendas.xlsx` → Arquivo principal contendo o dashboard
- `README.md` → Documentação do projeto

## 📊 Dados Utilizados
Os dados utilizados incluem:
- Data da venda
- Produto
- Categoria
- Quantidade vendida
- Receita
- Região ou vendedor

Os dados foram organizados em tabelas dentro do Excel e utilizados para criação de:
- Tabelas Dinâmicas
- Gráficos (barras, linhas e pizza)
- Segmentações (Slicers)

## ▶️ Como Reproduzir
1. Abra o arquivo `dashboard_vendas.xlsx`
2. Atualize os dados (se necessário)
3. Utilize os filtros interativos para explorar os dados
4. Analise os gráficos e indicadores

## 🛠️ Tecnologias Utilizadas
- Microsoft Excel
- Tabelas Dinâmicas
- Gráficos
- Segmentação de Dados

## 📈 Possíveis Melhorias
- Integração com Power BI
- Automação com VBA ou Python
- Atualização automática de dados
